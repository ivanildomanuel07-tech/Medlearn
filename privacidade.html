Política de Privacidade

Como recolhemos, usamos e protegemos os teus dados pessoais.

Última actualização: Abril de 2026

Dados que recolhemos

Ao criares uma conta no MedLearn, recolhemos os seguintes dados:

  • Nome completo
  • Endereço de email
  • Curso e ano académico
  • Preferências de estudo (definidas durante o onboarding)
  • Flashcards criados e progresso de estudo

Nunca recolhemos dados de pagamento, documentos de identificação, ou qualquer informação sensível não listada acima.

Como usamos os teus dados

Os teus dados são usados exclusivamente para:

  • Criar e gerir a tua conta na plataforma
  • Personalizar o conteúdo de estudo com base no teu curso e ano
  • Guardar o teu progresso e flashcards entre sessões
  • Melhorar a experiência da plataforma

Não usamos os teus dados para publicidade, nem os vendemos ou cedemos a terceiros para fins comerciais.

Armazenamento e segurança

Os teus dados são armazenados de forma segura através do Supabase, uma plataforma de base de dados com encriptação em repouso e em trânsito (HTTPS/TLS).

As passwords são geridas pelo sistema de autenticação do Supabase, que utiliza encriptação bcrypt — nunca armazenamos passwords em texto simples.

Partilha de dados

Os teus dados não são partilhados com terceiros, com excepção dos seguintes prestadores de serviço essenciais ao funcionamento da plataforma:

  • Supabase — armazenamento de dados e autenticação
  • Vercel — alojamento do site
  • Anthropic — processamento de perguntas enviadas à IA (apenas o texto da pergunta, sem dados pessoais)

Os teus direitos

Tens o direito de:

  • Aceder aos teus dados pessoais
  • Corrigir dados incorrectos
  • Solicitar a eliminação da tua conta e todos os dados associados
  • Exportar os teus dados

Para exercer qualquer um destes direitos, entra em contacto connosco através do email abaixo.

Cookies

O MedLearn utiliza apenas cookies essenciais para manter a tua sessão activa. Não utilizamos cookies de rastreamento ou publicidade.

Menores de idade

O MedLearn destina-se a estudantes universitários de medicina. Não recolhemos intencionalmente dados de menores de 18 anos. Se tomares conhecimento de que um menor criou uma conta, contacta-nos para que possamos eliminar os dados.

Alterações a esta política

Podemos actualizar esta política ocasionalmente. Quando o fizermos, actualizamos a data no topo desta página. Em caso de alterações significativas, notificamos os utilizadores por email.
